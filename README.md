# 匿名的砌砖人 / The Anonymous Bricklayer

> 神学微游戏 — 在巴别塔的阴影下，于沉默中砌砖。

## 试玩

**在线试玩**: [https://dengxiaocheng.github.io/TheologyGame-AnonymousBricklayer/](https://dengxiaocheng.github.io/TheologyGame-AnonymousBricklayer/)

## 本地运行

```bash
# 方式一：直接打开
open index.html          # macOS
xdg-open index.html      # Linux

# 方式二：本地服务器（推荐）
npx serve .
# 或
python3 -m http.server 8000
```

## 开发

运行语法检查：

```bash
node --check js/main.js
```

## 项目说明

本仓库为 Anonymous Bricklayer 神学微游戏的公开入口。游戏主题围绕巴别塔叙事，玩家在言语混乱的背景下以砌砖工的身份探索恩典与沉默。
